**1 功能**
使用环信SDK，来实现IM功能

**2 依赖的Gem**
``ProMotion``
``bubble-wrap``
``motion-cocoapods``
``cocoapods``
``nano-store``

**3 依赖的Cocopods**
``NanoStore``


**4 安装方法**
安装相关的Gem
``bundle install``
安装Cocoapods
``rake pod:install``
运行程序
``rake``
