
**1 功能**
1 使用极光推送，来实现消息推送功能

**2 依赖的Gem**
``ProMotion``
``bubble-wrap``
``motion-cocoapods``
``cocoapods``
``nano-store``

**3 安装方法**
1、到极光推送网站(www.jpush.cn)申请帐号，并上传相应的证书
2、修改配置文件 /resources/PushConfig.plist
3 安装相关的Gem
``bundle install``
4 运行程序
``rake``
